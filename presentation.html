نظام مراقبة الخوادم 🚀
Server Alert Manager

Built with Bash & Python

"نظام ذكي، خفيف، وآمن لمراقبة موارد السيرفر وإرسال التنبيهات"

📌 إيه هو المشروع ده؟ (Objective)

الهدف الأساسي هو إننا نعمل Monitoring لموارد السيرفر الحساسة زي مساحة الديسك Disk Space والرامات RAM.

ولما الاستهلاك يعلى عن الحد المسموح، السيستم بيبعت Automated Alert (تنبيه أوتوماتيكي) على الإيميل عشان الـ SysAdmin يتدخل قبل ما السيرفر يقع.

🛠️ التقسيم المعماري (Architecture)

عشان نكتب كود نظيف (Clean Code)، طبقنا مبدأ Separation of Concerns (فصل المهام).

  • الجزء الأول (المراقب): سكريبت Bash خفيف جداً بيتكلم مع نظام التشغيل عشان يقرأ الموارد من غير ما يستهلك الـ CPU.
  • الجزء التاني (المرسل): سكريبت Python قوي في الشبكات، بياخد الرسالة من الـ Bash ويبعتها لـ Gmail عن طريق اتصال مشفر.

❌ ليه معملناش MTA على السيرفر؟

طريقة Postfix القديمة

كان ممكن نستخدم أمر mail في لينكس، بس ده بيحتاج نسطب Postfix MTA.

  • بيستهلك موارد السيرفر (RAM).
  • ممكن الهاكرز يستغلوه كـ Open Relay ويبعتوا منه Spam.
  • عشان الإيميل يوصل للـ Inbox، محتاجين إعدادات معقدة في الـ DNS زي (SPF, DKIM).

طريقة Python الذكية

استخدمنا مكتبة smtplib في بايثون.

  • السيرفر بتاعنا بيشتغل كأنه (Client) بيكلم سيرفرات جوجل الموثوقة مباشرة.
  • الإيميل بيوصل 100% للـ Inbox.
  • مفيش أي بورتات غريبة بتتفتح في جدار الحماية (Firewall).

🔍 تفصيل الـ Bash Script (المراقبة)

استخدمنا أدوات اللينكس الأساسية Coreutils عشان سرعتها الخرافية.


# استخدام df و awk لجلب مساحة الديسك كرقم صافي
CURRENT_DISK_USAGE=$(df / | grep / | awk '{ print $5}' | sed 's/%//g')

# استخدام free و awk لجلب الرامات الفاضية بالميجا بايت
FREE_RAM_MB=$(free -m | awk 'NR==2{print $4}')

شرح الكود: أمر df / بيجيب جدول المساحة، أمر awk بيقص العمود رقم 5 اللي فيه النسبة، وأمر sed 's/%//g' بيمسح علامة % عشان نتحصل على رقم صحيح نقدر نقارنه في الـ If Condition.

📨 تفصيل الـ Python Script (الإرسال)

بمجرد ما الـ Bash يكتشف خطر، بينادي البايثون ويديله الرسالة.


from email.message import EmailMessage
import smtplib, os

msg = EmailMessage()
msg.set_content(alert_payload)
msg['Subject'] = 'CRITICAL: Server Alert'

# Secure SSL connection on Port 465
with smtplib.SMTP_SSL('smtp.gmail.com', 465) as smtp:
    smtp.login(os.environ.get('GMAIL'), os.environ.get('APP_PASS'))
    smtp.send_message(msg)

استخدمنا كلاس EmailMessage عشان نعمل Formatting صح للهيدرز (To, From, Subject)، ده بيمنع جوجل إنها تصنفنا كـ Spam.

🛡️ تأمين النظام (Security Practices)

  • منع الـ Hardcoding: ممنوع نكتب الباسورد جوه الكود! حطيناه في ملف مخفي اسمه .env عشان الكود يقرأه وقت التشغيل بس.
  • كلمات مرور التطبيقات (App Passwords): جوجل منعت الـ Basic Auth، فعملنا App Password مخصص للسكريبت.. لو اتسرق مش هيقدر يفتح جوجل درايف بتاعك، آخره يبعت إيميل وبس، وتقدر تلغيه بضغطة.

⚙️ ملف الإعدادات (Configuration)

عشان نبني سيستم محترف، مبدأ الـ Maintainability بيقول متخليش الأرقام جوه الكود.

عملنا ملف اسمه monitor.conf حطينا فيه:

  • MAX_DISK_USAGE=80
  • MIN_FREE_RAM=500

كده أي SysAdmin يقدر يغير التنبيهات من ملف الإعدادات من غير ما يلعب في الـ Code الأساسي ويبوظه.
